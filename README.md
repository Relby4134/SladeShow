# SlideShow Builder 🎯

Профессиональное приложение для создания и демонстрации слайд-шоу с поддержкой анимации, аудио и графических наложений.

## 🎯 Описание проекта

**SlideShow Builder** - это гибкое и расширяемое приложение для создания профессиональных презентаций и слайд-шоу. Проект демонстрирует применение современных паттернов проектирования и чистой архитектуры на Java с использованием JavaFX.

### Ключевые особенности:
- Создание и управление слайдами
- Поддержка различных форматов изображений
- Наложение графических объектов (текст, фигуры)
- Аудио сопровождение для каждого слайда
- Анимация переходов между слайдами
- Сохранение и загрузка проектов
- Гибкая система рендеринга

## ✨ Функциональность

### Основные возможности:
- ✅ Навигация по слайдам (вперед/назад)
- ✅ Добавление/удаление слайдов
- ✅ Изменение порядка слайдов
- ✅ Анимация переходов (Fade, Slide, Zoom)
- ✅ Аудио сопровождение
- ✅ Графические наложения на слайды
- ✅ Сохранение/загрузка проектов в JSON
- ✅ Два режима отображения (Basic/Rich/Animated)

## 🎨 Паттерны проектирования

### 1. **Abstract Factory (Абстрактная фабрика)**
**Где используется:** `SlideComponentFactory` и его реализации
**Назначение:** Создание семейств связанных компонентов (рендерер + переход + статус-бар)

```java
public interface SlideComponentFactory {
    SlideRenderer createRenderer();
    TransitionEffect createTransition();
    StatusBarView createStatusBar();
}
Реализации:

BasicSlideFactory - простые компоненты

RichSlideFactory - расширенные компоненты

AnimatedSlideFactory - анимированные компоненты

2. Iterator (Итератор)
Где используется: SlideIterator, SlideShowIterator
Назначение: Навигация по коллекции слайдов
public interface SlideIterator {
    boolean hasNext();
    boolean hasPrev();
    Slide next();
    Slide prev();
    Slide first();
    Slide last();
}
3. Builder (Строитель)
Где используется: Внутренний класс Slide.Builder
Назначение: Поэтапное создание сложного объекта Slide
Slide slide = new Slide.Builder(image, "path/to/image")
    .title("Мой слайд")
    .notes("Важные заметки")
    .animation(AnimationSettings.fadeIn(1000))
    .audio(new AudioTrack("audio.mp3", "Песня"))
    .addGraphic(new GraphicObject(...))
    .build();
4. Strategy (Стратегия)
Где используется: Рендереры, переходы, статус-бары
Назначение: Инкапсуляция различных алгоритмов

5. MVC (Model-View-Controller)
Где используется: Вся архитектура приложения

Model: Domain классы

View: JavaFX компоненты

Controller: SlideController
